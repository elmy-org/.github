# ELMY Learning Solutions

**Learning solutions built for the MENA region.**

We design and operate software that helps institutions teach, examine, and credential at scale — with first-class support for Arabic and the workflows our customers actually use.

## What we build

- **ELMY LMS** — our flagship learning management platform. Course delivery, syllabus, attendance, announcements, and a deeply-featured exam engine with proctoring, grading, and question banks. Django + DRF on the backend, Angular on the web, Ionic + Capacitor on mobile.
- **Question Editor** — a focused authoring tool for building and maintaining large question banks.
- **Customer-specific deployments** — tailored programs for organizations running their own examination and learning operations.

## Public projects

- [**GIFT-grammar-PEG.js**](https://github.com/elmy-org/GIFT-grammar-PEG.js) — a PEG.js grammar for the GIFT quiz format, used to parse and round-trip questions between authoring tools and the exam engine.
